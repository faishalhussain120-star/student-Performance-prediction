\# 🎓 Student Performance Prediction App



This is a Machine Learning web application built using Python and Streamlit.



\## 📌 Project Description

This app predicts student performance based on:

\- Study Hours

\- Attendance

\- Previous Marks

\- Sleep Hours

\- Assignments Completed



The model was trained using a machine learning algorithm and saved as model.pkl.



\## 🛠 Technologies Used

\- Python

\- Pandas

\- Scikit-learn

\- Streamlit

\- Joblib



\## 🚀 How to Run the Project



1\. Install requirements:

pip install -r requirements.txt



2\. Run the app:

streamlit run app.py



\## 📂 Project Files

\- app.py → Streamlit UI

\- train\_model.py → Model training script

\- model.pkl → Trained ML model

\- requirements.txt → Dependencies list

\- README.md → Project documentation
